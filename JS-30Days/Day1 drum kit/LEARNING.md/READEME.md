# 🥁 Drum Kit Project

## 📚 Today I Learned
Today I learned how to create a drum set using keyboard input.

The main idea was:
- Map each key press to a specific number
- Assign each number to a drum sound
- Play the corresponding sound when the key is pressed

On top of that, I also:
- Added click functionality so sounds play when buttons are clicked
- Implemented a custom drumstick mouse cursor for a more immersive experience

## 🚧 Learning Curve
One of the trickiest parts was positioning the drumstick cursor correctly.

I had to make sure that:
- The mouse click action aligns perfectly with the **tip of the drumstick**
- This made the interaction feel more natural and visually accurate

## 🎯 Outcome
- Interactive drum kit using keyboard and mouse
- Enhanced UI with a custom cursor
- Better understanding of event handling and mapping inputs to actions
